# People Counter System

A radar-based people counting system developed using **ESP32** and **Python** for real-time indoor occupancy monitoring.

---

## Project Overview

The purpose of this project was to develop an embedded IoT solution capable of detecting and counting people entering and leaving a room using radar technology.

The system combines embedded hardware, real-time communication and software visualization to provide accurate occupancy data.

---

## Features

- Radar-based people detection
- IN / OUT counting logic
- ESP32 microcontroller
- Python application
- Real-time visualization
- MQTT communication
- Data processing
- Technical documentation

# People Counter System

An embedded IoT system developed to detect and count people entering and leaving an area using millimeter-wave radar technology. The project combines embedded hardware, real-time data processing and a desktop application to provide accurate occupancy monitoring.

## Project Overview

The purpose of the project was to develop a reliable people counting system capable of detecting movement and calculating the number of people entering and exiting a monitored area. The system was designed as part of a collaborative engineering project where hardware, embedded software and desktop software were integrated into a complete solution.

The project provided practical experience of working with embedded systems, technical documentation, data collection and software development within a multidisciplinary team.
