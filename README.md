# People Counter System

A radar-based people counting system developed using **ESP32** and **Python** for real-time indoor occupancy monitoring.

---

## Project Overview

The purpose of this project was to develop an embedded IoT solution capable of detecting and counting people entering and leaving a room using radar technology.

The system combines embedded hardware, real-time communication and software visualization to provide accurate occupancy data.

---

## Features

- Radar-based people detection
- IN / OUT counting logic
- ESP32 microcontroller
- Python application
- Real-time visualization
- MQTT communication
- Data processing
- Technical documentation
